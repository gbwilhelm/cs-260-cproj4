About this project

This is a simple banking-management software with both front-end and back-end functionality, intended for use by bank employees; hence, there is no user authentication needed.
Currently, one can create a new client by filling in the appropriate input fields and then edit the information, which includes performing a withdrawal or deposit. A transaction history is kept for each client, and there are some fail-safes implemented to prevent illegal transactions or the creation of an invalid client. This lab can easily be extended to use a more detailed database entry for each client, such as storing their bank card information, Social Security number, phone number, co-account holders, or any number of relevant features.
I decided on using a modern, sleek, black and white design for this site to reflect the professional atmosphere that a bank would have.
